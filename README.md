## Resumo do curso

Durante o curso foram vistos:

* O mundo dos jogos [link dos slides](http://pt.slideshare.net/pdjkleber/o-mundo-do-desenvolvimento-de-jogos-61824823)
* Introdução a Unity 5.3.5
